# Project Dating App

Project from the Data Analyst course by Karpov.Courses.
- Exploratory Data Analysis
- A/B Testing Result Analysis
- Selecting metrics
- Hypothesis testing
- Calculating statistical significance
- Student's t-test, Welch's t-test, Chi-squared test
